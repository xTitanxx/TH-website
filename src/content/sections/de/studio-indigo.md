Ein eigenständiges Studio im Erdgeschoss, frisch renoviert. Ein Raum — Bett, kleine Küche, Bad hinter einer Schiebetür. Ebenerdig von der Straße: keine Stufe, keine Schwelle.

Gedacht für eine bis zwei Personen. Eine Woche, ein Monat, ein Winter. Ein ruhiger Arbeitsort oder einfach eine Art, langsamer zu werden. Gleiche Materialien, gleicher Blick, gleiche Textilien wie oben — in kleinerem Format.
